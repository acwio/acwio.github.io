Description

Software libraries (e.g., PyTorch, TensorFlow, and scikit-learn) power the world of machine learning. These libraries are used extensively as they provide out-of-the-box implementations for learning methods, making them usable to amateurs and experts alike. In this assignment, you will write the foundation for your own software library that will be used to implement and evaluate learning methods for the remainder of the Fall 2020 term.
